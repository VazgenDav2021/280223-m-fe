1. Для создание html документа кликаем на кнопку "New File...". После пишем анзвание документа. Обязательно после точки написать html чтобы у нас был правильный формат документа. До точки можно писать что угодно
2. Во время измения документа, надо сохранить изменения через ctrl/cmd + s, и в
   браузере сделать рефреш страницы (ctrl/cmd + r)

3. HTML состоит из элементов (теги). DOCTYPE, html, head, meta, title, body все являются тегами

4. h1 - h6. Это у нас заголовки разных уровний, самый большой h1, самый манленький h6.

5. p - просто параграф. Для добавления текста.

6. a - гиперссылка, которая переадрисовывает клиента по опредлелному пути. Путь указывается в "href". href это атрибут

7. Теги имеют атрибут. Атрибут это дополнительно свойстов для эелемнта.
   Есть атрибуты которые принадлежат только одному тэгу

8. img - служит для отаброжения фотографии. Имеет 2 своих атрибута.
   src - в него должный положыть ссылку фотографии, alt в него пишем текст.
   Текст alt-а отображается когда фотография не работает. Обязательно
   заполняем alt, для повышения SEO.

9. У тега img есть 2 атрибута widht и height. Они отвечают за высоту и ширину эелемта

10. Разделяют блочные и строчны элементы. Блочныйе (p, h) занимают всю ширину родителя.
    Строчные (img, a) занимают столько место сколько они есть
11. Разделяют теги на отркывающеяся и закрывающееся (p, a,h1) и
    отркывающеяся и сразу закрывающееся (br, img).

12. br переикыдавет эелемнт на новую строку

13. style - отвечает за применения стилей к элементу (шрифт, размер, цвет и тд)

14. text-aling: center, right, left(можно не писать так как делается по дефолту). Относится к блочным эелементам

15. color: red, green, #000, rgb (255,100,99). Задает цвет. Можно давать всем эелементам не зависимости блочный или строчный

16. background-color: red, green, #000, rgb (255,100,99). Задает задний фон. Можно давать всем эелементам не зависимости блочный или строчный

17. font-size: 10px 20px etc. Задает размер шрифта. Можно давать всем эелементам не зависимости блочный или строчный.

18. font-family: название шрифта. Есть несколько шрифтов в нашем ПК, если нужен кастомный можем скачать из интернета.

19. font-weight: жирность шрифта (100-900). Bold, italic, bolder,

20. Inspect element. Очень полезная технология для работы с веб сайтами и приложениями.Полезен для быстых правок и просмотра результата. После рефреша стрницы измения обнуляются.

21. У каждого элемента (тега) есть дефолтные стили, который задает браузер

22. Для того чтобы работать с CSS-ом надо его подключить к HTLM
    документу. Через тег link

23. Есть 3 способа задавания стилей:
    1: В HTML-е через(style).
    2: Внутри документа css, с привязкой к html
    3: Внутри тега head написать тег style и в нем писать как в css документе

24. С помощю Live Server мы можем делать изменения и после их сохранения, они выводится на экране (не надо делать рефреш)

25. div - является универсальным блочным элементом и предназначен для группирования элементов. Для того чтобы он был виден надо удоволетворять 2 условия.
    1: В нем должно что быть
    2: Он должен иметь ширину или высоту

26. border: none, "1px solid red". Задает границы для элемента

27. border-radius: 20px 20% etc. Округляет края у элемента

28. Для того чтобы у нас был круг:
    1: Стороны должны быть ровны (width height)
    2: border-radius = width/2 или height/2

29. У тега а есть атрибут target="\_blank". Если мы хотим сделать так чотбы при клике на гиперссылку у нас оставалсь текущая стрница
    и отркывалась новая страница с слыкой мы моежем это использовать
